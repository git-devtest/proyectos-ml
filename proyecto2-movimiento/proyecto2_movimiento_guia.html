Proyecto 2 — Detector de movimiento en tiempo real

Detecta zonas en movimiento comparando frames consecutivos. Implementa dos métodos clásicos de sustracción de fondo: diferencia absoluta (AbsDiff) y modelo gaussiano adaptativo (MOG2).

Idea central del proyecto

🎯 Sustracción de fondo

Técnica base de vigilancia: si el fondo es estático, cualquier diferencia entre frames es movimiento. El resultado es una máscara binaria donde blanco = movimiento, negro = fondo.

📐 Contornos sobre máscara

Una vez tenemos la máscara, findContours extrae las regiones blancas como polígonos. Filtramos por área mínima y dibujamos bounding boxes sobre el video original.

🔧 Morfología (dilate)

La máscara cruda tiene huecos y ruido. dilate expande las zonas blancas, uniendo fragmentos del mismo objeto y rellenando huecos internos antes de buscar contornos.

🎚️ Trackbar en tiempo real

createTrackbar agrega un slider nativo a la ventana OpenCV. Permite ajustar la sensibilidad sin tocar el código ni reiniciar el programa.

La franja verde muestra los píxeles donde la diferencia supera el umbral. Con umbral bajo se detectan cambios sutiles; con umbral alto solo cambios bruscos.

Relación con Proyecto 1: usamos exactamente el mismo loop de captura. Lo único nuevo es el procesamiento que ocurre entre cap.read() y imshow() — el frame pasa por el pipeline de detección antes de mostrarse.

Pipeline completo — paso a paso

Frame BGR → cvtColor GRAY → GaussianBlur → absdiff / MOG2 → threshold → dilate → findContours → filtrar área → boundingRect

Haz clic en cada paso para ver el detalle

¿Por qué convertir a gris antes de absdiff?

OperaciónCanalesComparaciones por píxelVelocidad
absdiff sobre BGR33 (B, G, R)Más lenta
absdiff sobre GRAY11~3× más rápida

Para una webcam de 640×480: BGR procesa 921,600 comparaciones por frame vs 307,200 en gris. A 30fps la diferencia es significativa.

Nota sobre MOG2: no necesita la conversión a gris — trabaja directamente con el frame BGR y extrae más información de color para separar fondo de primer plano. El paso cvtColor solo aplica al método AbsDiff.

Script completo

Estructura: mismas secciones numeradas que el Proyecto 1. Las secciones nuevas son la 5 (cálculo de máscara), 6 (morfología), 7 (contornos) y 9 (HUD con overlay semitransparente).
proyecto2_movimiento.pypython
import cv2
import numpy as np
from datetime import datetime

# ── 1. Abrir cámara ───────────────────────────────────────────────
cap = cv2.VideoCapture(0)
if not cap.isOpened():
    print("Error: no se pudo abrir la cámara")
    exit()

# ── 2. Configuración inicial ──────────────────────────────────────
metodo_activo   = "absdiff"
frame_anterior  = None
eventos_totales = 0

mog2 = cv2.createBackgroundSubtractorMOG2(
    history=500, varThreshold=50, detectShadows=False
)

# ── 3. Ventana y trackbar de sensibilidad ─────────────────────────
VENTANA = "Proyecto 2 - Detector de movimiento"
cv2.namedWindow(VENTANA)  # DEBE ir antes de createTrackbar
cv2.createTrackbar("Sensibilidad", VENTANA, 1500, 10000, lambda x: None)

# ── 4. Loop principal ─────────────────────────────────────────────
while True:
    ret, frame = cap.read()
    if not ret: break

    frame = cv2.flip(frame, 1)
    visualizacion = frame.copy()
    area_minima = max(cv2.getTrackbarPos("Sensibilidad", VENTANA), 100)

    # ── 5. Calcular máscara de movimiento ─────────────────────────
    if metodo_activo == "absdiff":
        gris = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)
        gris = cv2.GaussianBlur(gris, (21, 21), 0)
        if frame_anterior is None:
            frame_anterior = gris
            continue
        diferencia = cv2.absdiff(frame_anterior, gris)
        _, mascara = cv2.threshold(diferencia, 25, 255, cv2.THRESH_BINARY)
        frame_anterior = gris
    else:  # mog2
        mascara = mog2.apply(frame, learningRate=-1)
        _, mascara = cv2.threshold(mascara, 200, 255, cv2.THRESH_BINARY)

    # ── 6. Limpiar máscara con morfología ─────────────────────────
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))
    mascara = cv2.dilate(mascara, kernel, iterations=2)

    # ── 7. Encontrar contornos ────────────────────────────────────
    # OpenCV 4.x: devuelve 2 valores (contornos, jerarquía)
    contornos, _ = cv2.findContours(
        mascara, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE
    )
    hay_movimiento = False

    for contorno in contornos:
        if cv2.contourArea(contorno) < area_minima:
            continue
        hay_movimiento = True
        x, y, w, h = cv2.boundingRect(contorno)
        cv2.rectangle(visualizacion, (x, y), (x+w, y+h), (0, 255, 0), 2)
        cv2.putText(visualizacion, f"{int(cv2.contourArea(contorno))}px2",
                    (x, y-8), cv2.FONT_HERSHEY_SIMPLEX, 0.45, (0,255,0), 1)

    if hay_movimiento: eventos_totales += 1

    # ── 8. HUD semitransparente ───────────────────────────────────
    overlay = visualizacion.copy()
    cv2.rectangle(overlay, (0,0), (visualizacion.shape[1],40), (0,0,0), -1)
    cv2.addWeighted(overlay, 0.5, visualizacion, 0.5, 0, visualizacion)

    color_estado = (0,0,255) if hay_movimiento else (200,200,200)
    texto_estado = "MOVIMIENTO DETECTADO" if hay_movimiento else "Sin movimiento"
    cv2.putText(visualizacion, texto_estado,
                (10,26), cv2.FONT_HERSHEY_SIMPLEX, 0.65, color_estado, 2)

    alto = visualizacion.shape[0]
    cv2.putText(visualizacion, f"Eventos: {eventos_totales}",
                (10, alto-40), cv2.FONT_HERSHEY_SIMPLEX, 0.5, (180,180,180), 1)
    cv2.putText(visualizacion, "1=AbsDiff  2=MOG2  R=reset  S=captura  Q=salir",
                (10, alto-15), cv2.FONT_HERSHEY_SIMPLEX, 0.45, (120,120,120), 1)

    # ── 9. Mostrar ventanas ───────────────────────────────────────
    mascara_bgr = cv2.cvtColor(mascara, cv2.COLOR_GRAY2BGR)
    cv2.imshow("Mascara de movimiento", mascara_bgr)
    cv2.imshow(VENTANA, visualizacion)

    # ── 10. Teclado ───────────────────────────────────────────────
    tecla = cv2.waitKey(1) & 0xFF
    if   tecla == ord('q'):                   break
    elif tecla in (ord('1'), 177): metodo_activo = "absdiff"; frame_anterior = None
    elif tecla in (ord('2'), 178): metodo_activo = "mog2"
    elif tecla in (ord('r'), ord('R')):
        frame_anterior = None
        mog2 = cv2.createBackgroundSubtractorMOG2(history=500, varThreshold=50, detectShadows=False)
        eventos_totales = 0
    elif tecla in (ord('s'), ord('S')):
        nombre = f"movimiento_{datetime.now():%Y%m%d_%H%M%S}.png"
        cv2.imwrite(nombre, visualizacion)

# ── 11. Liberar recursos ──────────────────────────────────────────
cap.release()
cv2.destroyAllWindows()
print(f"Eventos detectados: {eventos_totales}")

Bugs conocidos y soluciones aplicadas

🐛 Bug crítico — findContours retorna distinto en OpenCV 3.x vs 4.x

Este es el bug más común al copiar código de Stack Overflow o tutoriales viejos.
diferencia entre versionespython
# OpenCV 3.x — RETORNA 3 VALORES (imagen, contornos, jerarquía)
_, contornos, _ = cv2.findContours(mascara, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# OpenCV 4.x — RETORNA 2 VALORES (contornos, jerarquía)  ← tu entorno
contornos, _ = cv2.findContours(mascara, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# Forma compatible con ambas versiones:
resultado = cv2.findContours(mascara, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
contornos = resultado[-2]  # penúltimo elemento — siempre son los contornos

Retos — de menor a mayor dificultad

A

Mostrar hora y fecha en el HUD

Agrega la hora actual en la esquina superior derecha de cada frame. Usa datetime.now().strftime("%H:%M:%S") dentro del loop. La fecha cambia cada frame — ideal para grabar evidencia.

B

Grabar video automáticamente al detectar movimiento

Crea un cv2.VideoWriter y escribe frames cuando hay_movimiento == True. Detén la grabación si no hay movimiento por más de 3 segundos. Usa cv2.VideoWriter_fourcc(*'XVID') para el codec.

C

Zona de interés (ROI) — detectar solo dentro de un área

Define una región (x1, y1, x2, y2) y aplica la detección solo dentro de ese recorte: mascara_roi = mascara[y1:y2, x1:x2]. Dibuja el rectángulo de la ROI en azul sobre el video.

D

Heatmap acumulativo de movimiento

Crea un array heatmap = np.zeros_like(frame, dtype=np.float32) y en cada frame suma la máscara: heatmap += mascara_bgr * 0.1. Usa cv2.applyColorMap(heatmap, cv2.COLORMAP_JET) para visualizarlo. Combínalo con el video con addWeighted.

E

Alerta visual persistente de 3 segundos

Guarda el time.time() cuando se detecta movimiento. Muestra "⚠ ALERTA" en rojo mientras time.time() - ultimo_evento < 3. Así la alerta no parpadea sino que se mantiene 3 segundos tras el último movimiento detectado.

💡 Consejo: el Reto B (VideoWriter) es el más valioso para vigilancia real. El Reto D (heatmap) es el más visualmente impresionante. Empieza por el A que es el más sencillo.
